# easytaxi-app
Ask for an Taxi from the menu bar

<img src="https://raw.githubusercontent.com/Urucas/easytaxi-app/master/screen.png" width="75%" />

# Build
```bash
git clone git@github.com:Urucas/easytaxi-app.git && cd easytaxi-app
npm i
npm start
```
